# Portfolio S5/S6 (site statique)

Ce dépôt contient un site web personnel (portfolio) prêt à déposer sur Git (ex : GitHub Pages).

## Structure
- `index.html` : accueil + photo + accès rapide
- `competences.html` : compétences + progression + preuves
- `projets.html` : projets illustrés + liens vers compétences (transitivité)
- `cv.html` : différentes versions de CV (PDF)
- `contact.html` : formulaire de contact + mailto

Assets :
- `assets/css/style.css`
- `assets/js/main.js`
- `assets/img/*`
- `assets/docs/*`

## À personnaliser
1. Remplacer `Prénom NOM`, email, LinkedIn, etc.
2. Remplacer les projets A/B/C/D par tes vrais projets + captures d’écran
3. Remplacer les PDF dans `assets/docs/`
4. Mettre ton vrai lien GitHub (rechercher `USERNAME`)

## Déployer sur GitHub Pages (rapide)
- Mettre ces fichiers à la racine du dépôt (branche `main`)
- Settings → Pages → Deploy from branch → `main` → `/ (root)`
